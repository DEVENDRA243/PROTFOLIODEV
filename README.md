# Portfolio - Devendra Kumar

This is my professional portfolio website built with React, TypeScript, and Vite.

## Features
- Interactive 3D robot
- Tech stack showcase
- Projects list
- Professional stats
- Terminal interface

## Tech Stack
- React
- TypeScript
- Vite
- Tailwind CSS
- Spline (3D)
- Lucide React (Icons)
- Framer Motion

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/DEVENDRA243/PROTFOLIODEV.git
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Build for production:
   ```bash
   npm run build
   ```
